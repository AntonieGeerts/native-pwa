Mandani Bay Community App API Tests

Note: These tests are designed to help diagnose API connectivity issues in the Mandani Bay Community App. Some tests are designed to run in a web browser, while others need to be run on a native device or emulator.

Test Results

After running the tests, please provide the following information:

  1. Which API URL worked correctly?
  2. Did the login functionality work with any of the URLs?
  3. Did the CapacitorHttp plugin work correctly?

This information will help us fix the API connectivity issues in the app.
